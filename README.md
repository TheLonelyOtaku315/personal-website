# Personal Portfolio Website

My personal portfolio website showcasing my projects and skills.

Visit: https://thelonelyotaku315.github.io/personal-website/

## Features

- Responsive design optimized for desktop, tablet, and mobile
- Dark/light theme toggle with smooth transitions
- Animated matrix-style background effect
- Welcome section with personal introduction
- About me section highlighting my background and skills
- Projects showcase
- Contact information and CV download
- Server-side rendering for improved performance and SEO

## Built With

- Angular 19
- TypeScript
- Express.js
- CSS3
- Flickity
